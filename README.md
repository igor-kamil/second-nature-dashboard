# second-nature-dashboard

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run electron:serve
```

### Compiles and minifies for production
```
npm run electron:build
```

```
npm run electron:build -- --win --x64
```

```
npm run lint
```
